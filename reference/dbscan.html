Example: Unsupervised clustering using DBSCAN

I was given the LPFC neural activity data collected from macaques in a decision-making task ( Yim et. al., 2019 ). We wanted to know how the activity of those neurons changed during task and whether some neurons responded in a similar manner. We had developed a minimal computational model that could "perform" the task as the animals did. We determined the essential features based on the model and extracted the four features from each LPFC neuron. Each neuron can be represented by one point in the 4D space.

Do they form clusters? They seem to. Do all neurons belong to a certain cluster? Maybe not because some points are just not close to any potential clusters. What clustering method shall we use?
The k-means clustering may not suit the purpose because all data points will be assigned to a cluster. We want to exclude outliers. In this case, we adopted density-based spatial clustering of applications with noise (DBSCAN). It is a density-based clustering non-parametric algorithm: given a set of points in some space, it groups together points that are closely packed together, marking as outliers points that lie alone in low-density regions (see ).
